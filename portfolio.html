2007
  • Distance (PHP) - view | source.

  • A Google maps mashup to calculate distances between two points. My goal was to use the highest level tools available to me. I use the Zend Framework to manage the backend, letting it take care routing and distance calculations. For the UI mechanics I use Javascript, Prototype, and JSON.

  • CalcCalc (C++) - source.

  • This was my entry for the OMGWTF programming competition. The object was to complete the simplest task (a 4 function calculator) in the most obtuse way possible. I use SendInput to mimic the CalcCalc user's input on the Windows Calculator. I then copy the answer, using GetMessage, to the CalcCalc result field.

2006
  • libgerpok (C++) - Source available upon request.

  • Libgerpok interfaces with Gerpok.com using a custom HTTP based protocol. The library manages user authentication, and communicating song start and finish events, which must meet criteria such as a minimum length of play-through, and well-formed and valid meta-data. It is also fault tolerant and able to operate off line. I achieved this by caching server communication and serializing song plays while off-line. The library utilizes the Loki, Boost, and cURL C++ libraries.

  • gpk2-winamp (C++) - source.

  • The Winamp client of libgerpok. I implemented a state machine to make sure only valid actions are taken by the user.

  • VocabBuilder (Java) - source.

  • I built this because I wanted a tool to build my vocabulary. Using Java, I parse a list of words, and output a formatted definition file. I utilize the Princeton Wordnet library as the program's dictionary.

2005
  • rQuip (Ruby) - source.

  • rQuip is a utility for solving the "crypto-quip" in the Sunday newspaper. It tries to decrypt a monoalphabetic substitution encrypted string by working off of known letter frequencies. It replaces the cyphertext alphabet with what it thinks it should be based off of statistical frequency. In order to test it I wrote a monoalphabetic character substitution encryption class called Quip.

  • rSort (Ruby) - source.

  • rSort is a radix sort implementation using Ruby. While contemplating binary numbers, it occurred to me that I could sort them by recursively comparing their bits from highest to lowest order. As I later learned, this principal is called a most significant bit radix sort.
